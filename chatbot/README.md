# Gemini Chatbot - React + Vite

A modern chatbot application built with React, Vite, and Bootstrap, powered by Google's Gemini 2.5 Flash model.

## Features

- 💬 Real-time chat interface with message history
- 🤖 Integration with Gemini 2.5 Flash API
- 🎨 Beautiful Bootstrap UI with custom styling
- 🌓 Dark mode support with toggle
- 📱 Fully responsive design with sidebar
- 💾 Multiple conversation support with delete option
- ⌨️ Keyboard shortcuts (Enter to send, Shift+Enter for new line)
- ⏹️ Stop generation feature
- 🎯 Markdown support for AI responses
- ⚡ Fast performance with Vite

## Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Google Gemini API key
